# Bandit Level 3 → 4

## Objective
Find the password in a hidden file in the `inhere` directory.

## Tools Used
- `ls -a`
- `cd`

## Process
1. Navigated to the directory: `cd inhere`
2. Listed all files (including hidden ones): `ls -a`
3. Read the hidden file: `cat .hidden`

## What I Learned
- Hidden files in Linux start with a `.` and require `ls -a` to be seen.
