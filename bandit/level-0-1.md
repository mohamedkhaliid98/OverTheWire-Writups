# Bandit Level 0 → 1

## Objective
Find the password stored in a file called `readme` in the home directory.

## Tools Used
- `ssh`
- `ls`
- `cat`

## Process
1. Connected to the server: `ssh bandit0@bandit.labs.overthewire.org -p 2220`
2. Listed the files: `ls`
3. Read the file: `cat readme`

## What I Learned
- How to connect to remote servers via SSH.
- Basic navigation and file reading.
